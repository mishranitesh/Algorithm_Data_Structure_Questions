STEPS TO COMPILE, RUN and CLEAN THE PROGRAM :-
------------------------------------------------

1. Please use below command to execute "PART-1 - Strassen's Matrix Multiplication" (compile, run and clean):
	make -s strassen_all

2. Please use below command to execute "PART-2 - Large Integer Multiplication" (compile, run and clean):
	make -s lim_all

3. Please use below command to execute "PART-3 - Heap and Radix Sorting" (compile, run and clean):
	make -s sort_all
